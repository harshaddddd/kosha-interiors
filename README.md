# Kosha Interiors — Next.js Website

Pune's highest-rated interior design studio website.

## 🚀 Quick Deploy (No Local Setup)

1. Upload this folder to GitHub
2. Connect repo to vercel.com
3. Click Deploy — done!

## 💻 Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000

## 📸 Images

Add your photos to `/public/images/` — see the README.txt inside that folder.

## 📁 Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Vanilla JS (no heavy libraries)
